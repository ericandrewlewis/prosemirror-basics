ProseMirror Basics

A bare bones "editor"

An editor can be quite basic. Here is an editor that accepts text input and that's about it.

Key bindings for Delete and Enter are in a plugin.

Undo, redo and history are in a plugin.

The basic setup

Marijn put together a "basic setup," which includes what you might expect to find in a text editor: a schema, a toolbar menu, undo history, keyboard shortcuts and magic input rules.

The document structure

An editor's document is a recursive node tree.

Each node can have attributes (eg. a heading has an attribute describing its level).

Nodes can contain text. Text is split into separate nodes based on their applied marks (eg. strong, emphasized or linked text).
